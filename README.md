# Application-Level Keyboard Input Logger

## Description
This project demonstrates keyboard event handling and file logging by recording user input within a controlled application environment.  
The application logs keystrokes typed **only inside the program window**, ensuring user awareness and ethical implementation.

This project is created for educational and internship evaluation purposes.

---

## Objective
- To understand keyboard event handling
- To implement file I/O operations in Python
- To log user input with timestamps
- To follow ethical software development practices

---

## Features
- Logs keys pressed inside the application only
- Saves keystrokes with timestamps to a text file
- Simple and user-consented GUI
- No system-wide or background monitoring

---

## Technologies Used
- Python 3
- Tkinter (GUI)
- File Handling
